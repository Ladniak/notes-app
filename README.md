# Notes App

A web application for creating, editing, and organizing personal notes with a simple and intuitive interface.

## Features

- Create and delete notes
- Edit existing notes
- Responsive design for all devices

## Installation

### Clone the repository:

```sh
git clone https://github.com/Ladniak/notes-app.git
```

### Navigate to the project directory:

```sh
cd notes-app
```

### Open the project in your code editor:

```sh
code .
```

### Install dependencies:

```sh
npm install
```

### Start the development server:

```sh
pnpm run dev
```

## Usage

1. Open the application in your browser.
2. Create a new note using the "Add Note" button.
3. Edit or delete notes as needed.
4. Use the search bar to quickly find notes.
